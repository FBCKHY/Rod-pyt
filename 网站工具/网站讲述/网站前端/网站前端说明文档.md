# 企业网站前端项目详细说明文档

## 📋 项目概述

本项目是一个完整的企业官网前端解决方案，采用传统的静态网站技术栈，为荣典企业提供专业的在线展示平台。项目支持PC端访问，具备完整的企业信息展示、产品展示、新闻资讯、联系方式等核心功能。

### 🎯 项目特点

- **📱 多端适配**：支持PC端和移动端（规划中）
- **🎨 模块化设计**：按功能模块组织代码结构
- **🚀 高性能**：静态页面，加载速度快
- **🔧 易维护**：清晰的文件组织和代码结构
- **📈 SEO友好**：纯HTML结构，利于搜索引擎收录

### 快速部署

#### 1. 直接部署
```bash
# 将整个 web PC 目录上传到Web服务器根目录
cp -r "web Site/web PC"/* /var/www/html/
```

#### 2. 访问配置
```
主页访问：http://yourdomain.com/index.html
关于我们：http://yourdomain.com/pages/about.html
产品中心：http://yourdomain.com/pages/products.html
联系我们：http://yourdomain.com/pages/contact.html
```

#### 3. 服务器配置示例 (Nginx)
```nginx
server {
    listen 80;
    server_name yourdomain.com;
    root /var/www/html;
    index index.html;
    
    # 静态资源缓存
    location ~* \.(css|js|jpg|jpeg|png|gif|ico|svg)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
    
    # HTML文件缓存
    location ~* \.html$ {
        expires 1h;
        add_header Cache-Control "public";
    }
    
    # 404错误页面
    error_page 404 /404.html;
}
```

```javascript
// 组件示例：图片懒加载
function lazyLoadImages() {
    const images = document.querySelectorAll('[data-src]');
    const options = {
        threshold: 0.1,
        rootMargin: '50px 0px'
    };
    
    const observer = new IntersectionObserver((entries) => {
        entries.forEach(entry => {
            if (entry.isIntersecting) {
                const img = entry.target;
                img.src = img.dataset.src;
                img.removeAttribute('data-src');
                observer.unobserve(img);
            }
        });
    }, options);
    
    images.forEach(img => observer.observe(img));
}
```

## 📊 性能优化建议

- **格式选择**：
  - JPEG：照片和复杂图像
  - PNG：透明背景和简单图形
  - WebP：现代浏览器首选格式
  - SVG：矢量图标和简单图形

- **压缩策略**：
```bash
# 图片压缩工具推荐

- Squoosh (Google工具)
```

### 2. 代码优化
```css
/* CSS优化示例 */
/* 避免深层嵌套 */
.navigation-menu .menu-item .sub-menu .link { /* 避免 */ }
.nav-link { /* 推荐 */ }

/* 使用CSS变量 */
:root {
    --transition-speed: 0.3s;
}
.button {
    transition: all var(--transition-speed) ease;
```

### 3. 加载优化
```html
<!-- 关键CSS内联 -->
<style>
/* 首屏关键样式 */
.header { ... }
.hero-section { ... }
</style>

<!-- 预加载重要资源 -->
<link rel="preload" href="assets/fonts/main-font.woff2" as="font" type="font/woff2" crossorigin>

<!-- 延迟加载非关键CSS -->
<link rel="preload" href="assets/css/non-critical.css" as="style" onload="this.onload=null;this.rel='stylesheet'">
```
