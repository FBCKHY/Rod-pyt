## 📊 技术栈总结

### 后端技术
- **框架**: Node.js + Express
- **数据库**: PostgreSQL + Sequelize ORM
- **认证**: JWT + bcrypt
- **文件上传**: Multer
- **API文档**: Swagger
- **日志**: Winston

### 前端管理
- **框架**: Vue 3 + Vite
- **UI库**: Element Plus
- **状态管理**: Pinia
- **HTTP**: Axios
- **TypeScript**: 完整类型支持

### 前端网站
- **技术**: 原生JavaScript (ES6+)
- **UI**: Bootstrap 5
- **图标**: Bootstrap Icons + Font Awesome
- **动画**: AOS
- **HTTP**: Fetch API

## 💡 使用指南

### 启动系统

```bash
# 1. 启动后端服务
cd backend
npm run dev
# 运行在 http://localhost:3001

# 2. 启动后台管理
cd admin
npm run dev
# 运行在 http://localhost:3006

# 3. 启动前端网站
cd "web Site/web PC"
python -m http.server 8080
# 运行在 http://localhost:8080
```
