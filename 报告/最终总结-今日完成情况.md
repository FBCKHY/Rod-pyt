## ⚠️ 发现的配置问题

### API地址不匹配

**问题**: 
- 前端配置: `http://localhost:3000/api`
- 后端运行: `http://localhost:3001/api`

**解决方案**:
```bash
# 修改 admin/.env.development 第6行
VITE_API_URL = http://localhost:3001/api
```

或者修改后端端口为3000。
