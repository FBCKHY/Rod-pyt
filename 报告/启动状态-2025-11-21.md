## ⚠️ 如果遇到问题

### 问题1: 后端启动失败
**可能原因**:
- MySQL服务未启动
- 端口3001被占用
- 依赖未安装

**解决方案**:
```bash
# 检查MySQL
net start MySQL

# 检查端口占用
netstat -ano | findstr :3001

# 重新安装依赖
cd backend
npm install
```

---

### 问题2: 前端启动失败
**可能原因**:
- 端口3006被占用
- 依赖未安装

**解决方案**:
```bash
# 检查端口占用
netstat -ano | findstr :3006

# 重新安装依赖
cd admin
npm install
```

---

### 问题3: API请求失败
**可能原因**:
- 后端未启动
- API地址配置错误

**解决方案**:
1. 确认后端运行在3001端口
2. 检查 `.env.development` 配置
3. 重启前端服务
