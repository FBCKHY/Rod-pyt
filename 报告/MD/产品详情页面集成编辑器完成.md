## 🔧 添加更多可编辑元素

如果你想让更多内容可编辑，按照以下步骤：

### 方法1: 手动添加

#### 找到要编辑的元素
```html
<h4>蓝焰技术</h4>
```

#### 添加 data-editable 属性
```html
<h4 data-editable="feature-1-title">蓝焰技术</h4>
```

### 方法2: 批量添加

使用查找替换功能：

#### 产品特点标题
```
查找: <h4>蓝焰技术</h4>
替换为: <h4 data-editable="feature-1-title">蓝焰技术</h4>
```

#### 规格参数
```
查找: <td class="specs-value">
替换为: <td class="specs-value" data-editable="spec-value">
```

## ⚠️ 注意事项

### 重要提醒
1. **定期保存**
   - 每编辑几项内容后点击"保存修改"
   - 防止意外关闭页面导致丢失

2. **导出备份**
   - 完成编辑后导出JSON配置
   - 保存文件作为备份

3. **图片优化**
   - 上传前压缩图片
   - 建议大小不超过2MB
   - 推荐尺寸 1200x1200px

4. **浏览器兼容**
   - 推荐使用 Chrome、Firefox、Edge
   - 不支持 IE 浏览器
