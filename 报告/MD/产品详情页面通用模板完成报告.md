## 🎯 使用流程

### 方式1: 直接编辑HTML（推荐）
```
1. 复制 RD-001 文件夹
2. 重命名为新产品型号
3. 打开 HTML 文件
4. 查找并替换所有 【】 占位符
5. 上传图片到对应文件夹
6. 在浏览器中预览
7. 发布上线
```
**时间**: 15-30分钟

### 方式2: 使用JSON配置
```
1. 复制 产品配置模板.json
2. 填写产品信息
3. 使用脚本生成HTML（未来功能）
4. 预览和发布
```
**时间**: 10-15分钟

### 方式3: 后台管理系统（规划中）
```
1. 登录后台
2. 填写表单
3. 上传图片
4. 一键生成
```
**时间**: 5分钟
