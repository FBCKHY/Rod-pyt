## 🚀 立即开始

### 第一步：创建数据库
```bash
cd backend
npx prisma init
# 编辑 prisma/schema.prisma
npx prisma migrate dev --name init
```

### 第二步：开发API
```bash
mkdir -p src/routes src/controllers
```

### 第三步：开发前端
```bash
cd admin
mkdir -p src/views/product
```
