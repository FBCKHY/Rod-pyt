### ✅ 步骤1: 修改API配置（2分钟）

**文件**: `admin/.env.development`

**操作**:
```bash
# 打开文件
notepad admin/.env.development

# 找到第6行，将：
VITE_API_URL = http://localhost:3000/api

# 改为：
VITE_API_URL = http://localhost:3001/api

# 保存并关闭
```

**或者使用命令行**:
```powershell
# 在项目根目录执行
(Get-Content admin\.env.development) -replace 'localhost:3000', 'localhost:3001' | Set-Content admin\.env.development
```

## 🔍 可能遇到的问题

### 问题1: 后端启动失败

**症状**: 
```
Error: connect ECONNREFUSED
```

**原因**: MySQL服务未启动

**解决**:
```bash
# Windows: 启动MySQL服务
net start MySQL

# 或者检查MySQL是否运行
```

---

### 问题2: 前端启动失败

**症状**:
```
Error: Cannot find module
```

**原因**: 依赖未安装

**解决**:
```bash
cd admin
npm install
npm run dev
```

---

### 问题3: API请求失败

**症状**: 
```
Network Error 或 404
```

**原因**: API地址配置错误

**解决**:
1. 检查 `.env.development` 配置
2. 确认后端运行在 3001 端口
3. 重启前端服务

---

### 问题4: 页面空白

**症状**: 页面加载但没有数据

**原因**: 数据库没有数据

**解决**:
```bash
# 运行测试数据脚本
cd backend
node migrations/seed-test-data.js
```
