## 🐛 常见问题排查

### 问题1: 菜单没有汉化

**可能原因**:
- 浏览器缓存
- 语言设置不正确

**解决方法**:
```bash
1. 按 Ctrl + Shift + R 强制刷新
2. 清除浏览器缓存
3. 检查语言设置（右上角）
```

### 问题2: 编辑器页面空白

**可能原因**:
- 前端网站未启动
- URL配置错误

**解决方法**:
```bash
1. 确认前端网站运行在 http://localhost:8080
2. 检查 admin/.env.development 中的 VITE_EDITOR_URL
3. 查看浏览器控制台错误信息
```

### 问题3: 订单详情加载失败

**可能原因**:
- 后端API未启动
- 订单ID不存在

**解决方法**:
```bash
1. 确认后端运行在 http://localhost:3001
2. 测试API: curl http://localhost:3001/api/orders/1
3. 查看浏览器Network面板
```

### 问题4: 前端网站显示空白

**可能原因**:
- API配置错误
- 后端未启动
- CORS问题

**解决方法**:
```bash
1. 打开浏览器控制台查看错误
2. 确认后端API运行正常
3. 检查 api-config.js 中的 baseURL
4. 测试API: curl http://localhost:3001/api/products
```
