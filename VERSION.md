## 🔧 技术栈

### 后端
- **框架**: Express.js
- **数据库**: PostgreSQL
- **ORM**: Sequelize
- **认证**: JWT
- **文件上传**: Multer
- **日志**: Winston

### 前端管理
- **框架**: Vue 3
- **构建工具**: Vite
- **UI库**: Element Plus
- **状态管理**: Pinia
- **路由**: Vue Router

### 前端网站
- **框架**: 原生HTML/CSS/JS
- **UI**: Bootstrap 5
- **图标**: Font Awesome, Bootstrap Icons
- **动画**: AOS (Animate On Scroll)

## 🚀 快速开始

### 1. 克隆代码
```bash
git clone https://github.com/FBCKHY/Rod-pyt.git
cd Rod-pyt
git checkout v1.0-production-ready
```

### 2. 安装依赖
```bash
# 后端
cd backend
pnpm install

# 前端管理
cd ../admin
pnpm install
```

### 3. 配置环境
```bash
# 复制环境变量模板
cp backend/.env.production backend/.env
# 编辑.env文件,填入实际配置
```

### 4. 启动服务
```bash
# 后端
cd backend
pm2 start ecosystem.config.js

# 前端管理(开发模式)
cd admin
pnpm dev
```

## 🔐 安全建议

### 生产环境配置
1. **修改默认密码**
   - 数据库密码
   - JWT密钥
   - 管理员密码

2. **启用HTTPS**
   - 配置SSL证书
   - 强制HTTPS重定向

3. **配置防火墙**
   - 只开放必要端口 (80, 443, 22)
