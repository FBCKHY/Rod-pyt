# 英雄区块图片目录

此目录用于存放首页英雄区块（Hero Section）的轮播图片。

## 图片命名规范

- 按照 `slide-数字.jpg` 格式命名
- 例如：`slide-1.jpg`, `slide-2.jpg`, `slide-3.jpg`, `slide-4.jpg`

## 图片要求

- 推荐尺寸：1920 x 1080 像素（16:9比例）
- 文件格式：JPG 或 WEBP（推荐）
- 文件大小：建议不超过 300KB（经过优化压缩）
- 图片内容：应与轮播文字内容相匹配

## 图片优化

为了提高网站加载速度，请确保所有图片都经过适当的压缩和优化：

1. 使用现代图像格式（如WEBP）
2. 使用图像压缩工具（如TinyPNG, ImageOptim等）
3. 考虑为不同设备准备不同分辨率的图片（使用picture元素或srcset属性）

## 占位图片

目前使用的是占位图片，实际部署时请替换为高质量的产品或场景图片。
